To run the code you need:
- [Lua](https://www.lua.org/)
- A table.csv file containing books

When the program starts, you have the option to either enter "C", to create a book, "S" to search for a book, and "U" to update a book.

Create:
- First you enter the title
- Then you enter the author
- Then you enter the year it released

Search:
- You enter if you want to search for title, author, or year
- You enter either the title, author, or year, depending on what you entered
- It shows the book if it exists

Update:
- It shows you a list of all the books, you enter the number associated with the book
- You enter if you want to change the title, author, or year
- You enter what you want to change it to
